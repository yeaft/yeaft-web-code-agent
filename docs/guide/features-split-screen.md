# Split Screen

Open multiple conversations side by side — up to 3 panels at once.

## Features

- **Split from sidebar** — Click the split icon on any session to open it in a new panel
- **Independent panels** — Each panel is a fully independent conversation view with its own chat history
- **Active-panel focus** — Click a panel to set it as the active target for sidebar session clicks
- **Close individually** — Each panel has a close button; closing all panels returns to single-panel mode
- **Works with Crew** — Crew conversations can also be split into separate panels
