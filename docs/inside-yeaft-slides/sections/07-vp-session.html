Session

Session

A Session is the only orchestration unit. One VP gives a normal 1:1 code-agent conversation; multiple VPs create collaboration without inventing another mode.

Session 是唯一编排单元。一个 VP 就是普通 1:1 code-agent 对话；多个 VP 就是协作，不需要再发明另一种 mode。

VP

VP

A Virtual Person carries persona, model choice, tool access, memory scope, and sub-agent authority. It is not just a label; it changes how the loop is built.

Virtual Person 带有人格、模型选择、工具权限、记忆 scope 和子 Agent 权限。它不只是标签，而会改变一次 loop 如何构造。
