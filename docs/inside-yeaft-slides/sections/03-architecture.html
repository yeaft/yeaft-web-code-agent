03 · System map

A code agent is a controlled feedback loopCode Agent 是一个受控反馈循环

01

Input

User turn, attachments, mentions, selected VP.

用户输入、附件、提及对象、选中的 VP。

02

Context

Prompt, docs, memory recall, platform facts.

Prompt、项目文档、记忆召回、运行平台事实。

03

LLM

Adapter router selects provider, model, effort.

Adapter router 选择 provider、模型和推理强度。

04

Tools

Execute file, shell, web, skills, MCP, sub-agent calls.

执行文件、Shell、Web、技能、MCP、子 Agent 等调用。

05

Reflect

Fold long arcs, summarize, continue if needed.

折叠长工具弧、总结、必要时续写。

06

Persist

Conversation, raw outputs, traces, memory updates.

持久化对话、原始输出、trace 和记忆更新。
