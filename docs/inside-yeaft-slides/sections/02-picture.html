Yeaft is a Web UI plus a native Session engine. The important boundary is not "chat mode versus group mode"; it is one Session containing one or more VPs.

Yeaft 是 Web UI 加原生 Session 引擎。关键边界不是“聊天模式 vs 群组模式”，而是一个 Session 里有一个或多个 VP。
