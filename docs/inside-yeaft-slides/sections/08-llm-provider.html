Provider config

Provider 配置

Yeaft reads agent-local providers from ~/.yeaft/config.json: static API keys, dynamic credentials, base URLs, protocol, model overrides, context windows, and effort options.

Yeaft 从 ~/.yeaft/config.json 读取 agent-local provider：静态 key、动态凭证、base URL、protocol、模型覆盖、上下文窗口和 effort 选项。

Protocol adapters

协议适配器

The AdapterRouter maps model IDs to Anthropic Messages or OpenAI Responses protocol, with per-model overrides when one provider exposes mixed model families.

AdapterRouter 把模型 ID 映射到 Anthropic Messages 或 OpenAI Responses 协议；同一个 provider 暴露混合模型族时可以做 per-model 覆盖。

providers[] → AdapterRouter
  → protocol: anthropic | openai-responses
  → model effort / thinking / reasoning
  → stream text + thinking + tool_calls + usage
  → retry / fallback model when configured
