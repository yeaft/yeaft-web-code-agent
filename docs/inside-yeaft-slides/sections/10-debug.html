10 · Traceability

Debugging is part of the product, not an afterthought调试不是附属品，而是产品的一部分

Loop trace

循环 trace

Pre-query context, adapter stream, tool calls, usage, retries, compact, and continuation can be recorded and inspected.

pre-query 上下文、adapter stream、tool call、usage、重试、compact、续写都可以被记录和检查。

Raw outputs

原始输出

UI may show folded summaries, but raw tool output and debug traces can remain available inside the right storage boundary.

UI 可以显示折叠摘要，但原始工具输出和 debug trace 可以在正确存储边界内保留。

Agent liveness

Agent 活性

Background tasks and sub-agents expose status, logs, terminal notifications, and liveness counters instead of silent waiting.

后台任务和子 Agent 暴露状态、日志、终止通知和 liveness 计数，而不是静默等待。
