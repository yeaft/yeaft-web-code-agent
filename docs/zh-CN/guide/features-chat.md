# Chat 聊天

ChatGPT 风格对话界面，实时工具追踪，会话管理和文件上传。

![Chat](/images/zh-CN/chat.jpg)

## 功能

- **实时流式输出** — Claude 响应实时流式输出
- **工具可视化** — 可视化显示 Read、Edit、Bash 等工具操作，支持展开查看详情
- **斜杠命令** — 内置命令（`/model`、`/memory`、`/skills` 等），支持自动补全
- **`/btw` 侧边提问** — 在不打断当前任务的情况下快速追问
- **Sub-Agent 面板** — 实时监控和查看嵌套 Agent 工具调用
- **会话持久化** — 所有对话保存到 SQLite，可随时恢复
- **会话置顶** — 将重要对话固定在侧边栏顶部
- **文件附件** — 拖放上传文件和图片，作为上下文发送给 Claude
- **深色 / 浅色主题** — 一键切换主题
- **双语界面** — 中英文运行时切换
- **移动端适配** — 完整功能的移动端响应式布局
