# Crew 多角色协作

多角色 AI 团队协作，PM、开发者、审查者、测试者等角色协同完成 Feature 开发。

![Crew Features](/images/zh-CN/crew-features.jpg)

![Crew Feature Detail](/images/zh-CN/crew-feature-detail.jpg)

## 功能

- **自动任务路由** — 角色间通过 ROUTE 协议自动任务路由
- **Feature 进度追踪** — 实时状态面板，streaming 呼吸灯动画
- **角色分组消息** — 决策者消息在主流中直接显示，按角色分组
- **多 Agent 并行执行** — 多 Agent 跨 worktree 并行执行
- **智能完成检测** — Feature 完成检测，有新活动时自动重新激活
- **AskUserQuestion 交互卡片** — Agent 可在任务进行中向用户请求决策
- **Typing Indicator 健康监控** — 事件驱动状态：Agent 离线 / Session 丢失 / 正在压缩
