# 快捷键

Yeaft Web 端的键盘快捷键速查。

## 全局

| 快捷键 | 作用 |
| --- | --- |
| **Ctrl+/** （Cmd+/） | 显示快捷键帮助 overlay |
| **Esc** | 关闭弹窗 / 取消当前 modal |
| **Ctrl+,** （Cmd+,） | 打开设置 |

## 聊天输入框（Chat / Copilot / Yeaft 通用）

| 快捷键 | 作用 |
| --- | --- |
| **Enter** | 发送消息 |
| **Shift+Enter** | 换行 |
| **/** | 打开斜杠命令自动补全（仅 Claude Code） |
| **@** | 打开 mention 自动补全（Crew 选角色 / Yeaft Group 选 VP） |
| **Ctrl+V** （Cmd+V） | 粘贴文本或图片（图片自动成附件） |
| **↑** | 调出上一条已发送消息（草稿恢复） |

## 自动补全菜单

| 快捷键 | 作用 |
| --- | --- |
| **↑** / **↓** | 上下选项导航 |
| **Tab** | 选中并确认 |
| **Enter** | 选中并确认 |
| **Esc** | 关菜单 |

## 图片预览 overlay

| 快捷键 | 作用 |
| --- | --- |
| **Esc** | 关闭预览 |
| **←** / **→** | 上一张 / 下一张（多附件时） |
| 鼠标点暗色背景 | 关闭预览 |

## 文件浏览器（Workbench → Files）

| 快捷键 | 作用 |
| --- | --- |
| **Ctrl+P** （Cmd+P） | 快速打开文件搜索 |
| **Ctrl+Shift+P** | 命令面板 |
| **Ctrl+滚轮** | 调整字体大小 |
| **F2** | 重命名选中项 |
| **Delete** | 删除选中项（会弹确认） |
| **Enter** | 打开选中文件 |

## 文件编辑器（Workbench → Files）

| 快捷键 | 作用 |
| --- | --- |
| **Ctrl+S** （Cmd+S） | 保存文件 |
| **Ctrl+F** （Cmd+F） | 文件内查找 |
| **Ctrl+H** （Cmd+H） | 查找并替换 |
| **Ctrl+Z** / **Ctrl+Shift+Z** | 撤销 / 重做 |
| **Ctrl+/** | 注释 / 取消注释当前行 |

## 终端（Workbench → Terminal）

| 快捷键 | 作用 |
| --- | --- |
| **Ctrl+C** | 中止当前命令 |
| **Ctrl+D** | 关闭终端 / 退出 shell |
| **Ctrl+L** | 清屏 |
| **Ctrl+Shift+C** | 复制选中文本 |
| **Ctrl+Shift+V** | 粘贴 |
| **Ctrl+滚轮** | 字体大小 |

## 分屏 / 多面板

| 快捷键 | 作用 |
| --- | --- |
| **Ctrl+点击 sidebar session** | 在新面板打开（开启分屏） |
| 面板头部 **×** | 关闭面板 |

## Yeaft Group Mode

| 快捷键 | 作用 |
| --- | --- |
| **@** | 选择要 mention 的 VP |
| **Ctrl+Enter** | 发送给全员（忽略 mention，全 fan-out） |

## 提示

- macOS 用户把 **Ctrl** 替换成 **Cmd**（除明确写 Ctrl+Shift 的情况）
- 大部分快捷键参考 VS Code 习惯 — 你熟悉 VS Code，这里也熟悉
- 想 reset：刷新页面，所有 in-memory 快捷键状态都会回到默认
